# Contributing to Cellyn

First off, thanks for taking the time to contribute! 🎉

## Project Attribution

**Cellyn** is a WhatsApp bot built by [Liwirya](https://github.com/liwirya). The bot's **architecture and project structure** are inspired by [Katsumi](https://github.com/nat9h/Katsumi) by [nat9h](https://github.com/nat9h) — huge respect to them for the solid foundation.

However, **this README, documentation, plugin implementations, and all code modifications** are original work by the Cellyn team. If you're borrowing ideas from this repo, please give credit where it's due — both to Cellyn and Katsumi.

## How to Contribute

### Reporting Bugs

- Open an [Issue](https://github.com/liwirya/cellyn-whatsapp-bot/issues)
- Include steps to reproduce, expected vs actual behavior
- Add screenshots/logs if possible

### Suggesting Features

- Open an Issue with the `enhancement` label
- Describe the use case and proposed solution

### Pull Requests

1. Fork the repo
2. Create a feature branch: `git checkout -b feature/nama-fitur`
3. Commit with clear messages
4. Push and open a PR
5. Wait for review — we'll get back to you ASAP!

## Code Style

- Use **ES Modules** (`import/export`)
- Follow existing plugin structure
- Add JSDoc comments for complex functions
- Keep plugins focused — one responsibility per file
